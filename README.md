#  Mental-HealthCare-Chatbot
MindMate is a mental healthcare chatbot built with HTML, CSS, and Java, powered by the Gemini Flash API. It offers friendly conversations, mindfulness tips, and motivational resources to help users manage stress and anxiety. With a responsive design, secure backend, and smart AI, it acts as a supportive wellness buddy, not a therapy replacement.

#  Features

-   **Instant Responses:** Powered by Google's Gemini 1.5 Flash model for fast and coherent interactions.
-   **Empathetic Persona:** The AI is prompted to act as a supportive mental health companion.
-   **Responsive Design:** Clean and soothing UI/UX compatible with desktop and mobile devices.
-   **No Backend Required:** Runs entirely on the client-side using Vanilla JavaScript.
-   **Conversation History:** Maintains the context of the chat for a natural flow.

#  Tech Stack

-   **Frontend:** HTML5, CSS3
-   **Logic:** JavaScript (ES6+)
-   **AI Model:** Google Gemini API (Gemini 1.5 Flash)

#  Key Features

- Powered by Gemini 1.5 Flash: Utilizes Google's latest high-speed model for instant interactions without lag.
- Empathetic Companion: Specifically prompted to offer supportive, non-judgmental, and active listening responses.
- Context Awareness: Maintains chat history within the session so the AI remembers what you shared earlier.
- Soothing UI/UX: Designed with calming colors and a clean layout to ensure a stress-free user experience.
- Privacy-Focused: Runs client-side; no chat data is stored on external servers permanently.

#  Important Medical Disclaimer

This chatbot is an AI-based project developed for educational and supportive purposes only. It is **NOT** a substitute for professional medical advice, diagnosis, or treatment.

-   If you or someone you know is in crisis, please contact local emergency services or a suicide prevention hotline immediately.
-   The AI may occasionally generate incorrect information.
